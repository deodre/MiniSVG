# MiniSVG
Multimedia Project

Descriere: Construirea unui editor pentru grafică vectorială folosind elementul SVG.

*1p Suport pentru adăugare elemente geometrice de bază (linie, elipsă, dreptunghi)
*1p Suport pentru adăugare și editare text
*2p Suport pentru selecție / ștergere / mutare / modificare proprietăți
*1p Suport pentru efecte și transformări
*1p Redare de sunete asociate operațiilor
*2p Suport pentru poligoane / căi cu editarea ulterioară a punctelor
